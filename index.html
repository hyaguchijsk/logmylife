How to USE? (Food Section)

First of all, please input food information via "Add New Food", "Vendor", "Name", "Energy" is required, then click "Registration". then you can choose it from "What did you eat?". If you input "Vendor" and "Name" the same as exist item, then "Registration" will update information.

Please choose "What did you eat?" and when, if now, check "Now", else check "Set time as" and select time following pull down menu, then click "Registration".

You can show food history what did you eat in a day. Also you can travel your history via pull down menu.

まず，食べ物の情報を"Add New Food"から登録してください． "Vendor", "Name", "Energy"は必須項目です． "Registration"で登録できます． 登録した食べ物は"What did you eat?"から選択できます． "Vendor"と"Name"が登録されているものと同じ場合， "Registration"は情報の更新を行います．

"What did you eat?"から食べ物を選択し，時間を選びます． もし今であれば"Now"にチェックを入れてください． 時間を指定したい場合は"Set time as"にチェックを入れて その下のプルダウンメニューから日時を選び， "Registration"で登録を行なってください．

一日の食事の履歴を見ることができます． またプルダウンメニューから日付を選択し過去の履歴を見ることができます．
